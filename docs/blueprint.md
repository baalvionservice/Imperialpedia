# **App Name**: Imperialpedia

## Core Features:

- Homepage Display: Render the initial 'Imperialpedia — Financial Knowledge Platform' content on the root page, demonstrating basic UI and styling with Tailwind CSS integration.
- Modular Page Integration: Implement a foundational modular structure in the `/modules` directory to showcase how future content pages (e.g., glossary entries) would be integrated using the App Router.
- Mock Content API: Establish a basic mock API service structure within the `/services` directory to simulate data retrieval for financial content, preparing for future backend integration.
- Base SEO Configuration: Configure essential Next.js metadata and implement a base SEO structure within the root layout for optimal search engine indexing and discoverability of content.
- Root Layout & Navigation: Implement the global root layout using Next.js App Router, providing a consistent structural foundation for all application pages and integrating global styles.
- AI-Assisted Content Outline Tool: Utilize generative AI to propose initial article outlines and suggest key topics for new financial knowledge content, serving as a tool for content creators and SEO optimization.

## Style Guidelines:

- A dark color scheme conveys sophistication and focus. Primary accent color: A rich, distinguished violet (#8272F2) for interactive elements and highlights, suggesting insight and quality. Background color: A very dark, subtle purplish-gray (#1C1822) to provide a refined backdrop. Accent color: A vibrant, clear sky blue (#69B9FF) for secondary actions or contrasting information.
- Headline font: 'Playfair' (serif) for an elegant, editorial, and high-end feel. Body font: 'PT Sans' (sans-serif) for its modern readability and warmth, ensuring comfortable consumption of extensive financial knowledge content.
- Utilize a consistent set of clean, vector-based icons with a professional and precise aesthetic. Prioritize outline or minimalist filled styles for clarity across diverse content modules.
- Employ a spacious, grid-based layout that prioritizes content readability and visual hierarchy. Utilize ample whitespace to reduce cognitive load and define distinct modular sections for financial topics.
- Implement subtle and purposeful micro-animations for user feedback (e.g., button hovers, content loading states) and smooth transitions between page sections, enhancing perceived performance and polish.
